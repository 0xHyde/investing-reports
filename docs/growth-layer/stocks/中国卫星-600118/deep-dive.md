# 中国卫星（600118）深度分析报告

> 报告日期：2026年4月27日  
> 所属赛道：低轨卫星 · 卫星总装制造  
> 市场标签：国家队 / 航天科技集团核心上市平台

## 一、叙事书

### 1.1 产业逻辑：低轨卫星进入"星座竞赛"时代

全球低轨卫星（LEO）正处于新一轮军备竞赛期。SpaceX星链已部署超7000颗卫星，中国"千帆星座"、"星网工程"加速推进，规划卫星数量均超过万颗。卫星制造是整个产业链的"心脏环节"——星座部署速度取决于卫星批产能力，而总装制造是产能瓶颈中的瓶颈。

**产业核心逻辑链条：**
- 政策端：卫星互联网纳入新基建，"十四五"航天规划明确大规模星座部署
- 需求端：6G空天地一体化、国防安全、偏远地区覆盖三大刚需驱动
- 供给端：卫星从"科研定制"向"工业批产"转型，成本下降是核心变量
- 时间窗口：2025-2028年是星座大规模发射的关键窗口期

### 1.2 竞争格局：国家队主导地位明确

中国卫星产业呈现"国家队主导、民企补充"的格局：

| 梯队 | 代表企业 | 定位 |
|------|---------|------|
| 第一梯队 | 中国卫星（航天五院）、航天科技/科工体系 | 卫星总装、核心载荷 |
| 第二梯队 | 银河航天、长光卫星、微纳星空 | 卫星研制、星座运营 |
| 第三梯队 | 众多零部件供应商 | 元器件、分系统 |

**中国卫星的独特卡位：**
- 国内唯一从事小/微小卫星研制及卫星应用设备生产的上市公司
- 背靠航天科技集团五院（中国空间技术研究院），承担国家队核心任务
- 在小卫星平台、通信/遥感/导航卫星总装领域市占率领先
- 深度参与星网工程、遥感卫星等国家重大专项

### 1.3 公司故事：从"航天贵族"到"批产先锋"

中国卫星的发展史是一部中国空间技术商业化的缩影：

- **基因优势**：脱胎于航天五院，技术底蕴深厚，参与过东方红系列、北斗导航、嫦娥探月等国家级工程
- **转型阵痛**：传统航天项目周期长、交付慢，面对星座时代的批产需求，公司正在经历从"单星定制"到"流水线批产"的痛苦转型
- **2025年业绩断崖**：营收51.6亿/-25.1%，利润仅0.28亿/-82.3%，反映出传统业务收缩与新业务爬坡的"青黄不接"
- **2026Q1积极信号**：营收4.42亿/+33.6%，毛利率回升至16.0%，订单回暖迹象初显

**核心看点**：星网工程进入密集发射期后，公司作为总装单位有望迎来订单爆发。若批产能力提升，业绩弹性巨大。

### 2.2 增长与盈利能力

**2025年报**：
- 营收同比-25.1%，利润同比-82.3% → 双杀格局
- 毛利率11.77%，低于高端制造业平均水平

**2026Q1**：
- 营收4.42亿/+33.6% → 订单端出现积极变化
- 利润-0.24亿（仍亏损）/-13.0% → 尚未扭亏
- 毛利率16.02% → 环比回升，结构改善
- ROE -0.38% → 一季度通常为淡季，但仍需观察

### 2.3 估值分析

当前PB 17.69x，显著高于：
- 军工行业PB均值（约3-5x）
- 高端装备制造业PB（约2-4x）

**估值逻辑**：市场给予极高溢价，定价的是"星网工程核心供应商"的期权价值，而非当前盈利能力。若星座建设不及预期，估值回归风险极大。

### 2.4 财务健康度

- 资产负债率：航天系企业通常稳健
- 现金流：军品回款周期较长，需关注经营性现金流
- 应收账款：航天项目占款问题需警惕

## 三、风险扫描

### 🔴 高风险项

1. **业绩兑现风险**：当前市值1,119亿对应2025年利润PE超4000倍，若星网订单延迟或批产不及预期，估值将剧烈回归
2. **竞争格局变化**：商业航天政策放开后，银河航天等民企快速崛起，可能侵蚀市场份额
3. **盈利能力薄弱**：毛利率不足12%，净利率接近0，商业模式尚未跑通

### 🟡 中风险项

4. **政策与订单波动**：航天项目受国家规划影响大，订单节奏不确定
5. **技术迭代风险**：卫星小型化、标准化趋势下，传统平台技术可能面临淘汰
6. **一季度亏损扩大**：Q1利润同比继续下滑，短期趋势未完全扭转

### 🟢 低风险项

7. **信用与治理风险**：航天科技集团背景，信用风险极低

### 策略建议

**短期（3-6个月）**：Q1营收回暖提供交易性机会，但一季度仍亏损，不宜追高。关注星网工程招标进展及半年报能否扭亏。

**中期（1-2年）**：若星网进入密集发射期，订单放量可能带动业绩反转。关键跟踪指标：新签订单金额、毛利率变化、单季度利润是否转正。

**长期（3年+）**：卫星批产能力决定长期价值。若能成功转型为"航天制造平台型企业"，当前高估值或可通过高增长消化。

### 仓位建议

- **保守型投资者**：回避。当前估值无安全边际，财务数据不支持。
- **成长型投资者**：可小仓位（1-2%）配置作为卫星互联网主题 exposure，需严格止损。
- **激进型投资者**：逢低布局，博弈星网订单爆发，但需承受高波动。
