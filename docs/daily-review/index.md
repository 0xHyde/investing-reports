# 📅 每日复盘

**这里是每日持仓复盘的存档。**

复盘遵循「六步法」框架：
1. **市场叙事** — 提炼当日市场主线故事
2. **资金流向** — 成交额、天量信号、主力动向
3. **逐只深度分析** — 核心层/成长层每只标的的市场叙事、老大的故事、是否通顺
4. **组合层面诊断** — 两层框架检查、集中度风险、交易行为审计
5. **经验沉淀** — 做对了什么、做错了什么、市场教会了什么
