# 投资哲学纲领

> 来源：`investment-philosophy` skill，具有最高优先级。

## 十大投资红线

1. **绝对不使用任何杠杆**。
2. **个股合计 ≤ 总资产 25%，单只个股 ≤ 4%**（小资金过渡期：个股合计≤40%，单只≤10%）。
3. **成长层 ≤ 总资产 25%**，利润必须向上流动至核心层，不得在成长层内滚存。
4. **核心层 ≥ 总资产 30%**，人类无权单方面削减。
5. **任何策略调整必须经过至少 7 天冷静期**，并由老大明确书面确认。
6. **年度储蓄率硬底线 ≥ 50%**，目标线 ≥ 60%。
7. **收入到账后先投资、后消费**。
8. Agent 不主动挖掘、推荐任何个股或成长层标的。
9. **信息节食**：不追踪短期预测、热点、股评。不盯盘、不看日线。
10. **情绪干预模式**：当老大表达想暂停定投/加仓/割肉时，Agent 不讨论行情，优先援引纲领条款并启动冷静期。

## 小资金过渡期条款

适用条件：总资产 < 50万。单只个股上限从4%放宽至10%，个股合计上限从25%放宽至40%。触发回归条件：总资产≥50万时自动回归标准版。

## 两层架构

- **核心层**（目标 ≥ 60%）：高息股个股 + 红利ETF + 全球宽基 + 政策基础
- **成长层**（目标 ≤ 25%）：为未来清晰叙事付费的进攻性配置

> 利润必须向上流动。成长层收益不得内滚。
