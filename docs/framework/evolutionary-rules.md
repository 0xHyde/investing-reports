# 框架演进规则 (Evolutionary Rules)

> 文件性质：进化规则  
> 生效条件：永久  
> 审批要求：需经老大明确书面确认

---

## 一、基本原则

1. **框架可进化，纲领不变**
   - 本框架允许根据实际情况修订，但《投资纲领》的 10 条核心约束（红线）任何时候都不得触动。
   - 框架修订只能调整"执行细则"和"比例分配"，不能动摇"风控原则"和"行为约束"。

2. **冷静期原则**
   - 任何对本框架的修订建议，必须经历 **至少 7 天冷静期**，由老大明确书面确认后才可生效。
   - 修订建议须以文本形式提出，不接受口头随意调整。

3. **版本化管理**
   - 每次修订后，需更新版本号（格式：YYYY.MM.DD-N）。
   - 所有历史版本保留，不得覆盖或删除。

## 二、小资金过渡期条款（临时例外）

> 启动日期：2025-04-19  
> 启动原因：总资产约 16.3 万，严格执行 4% 单只上陑导致交易成本过高、操作不具可行性  
> 性质：**临时性例外**，非永久修改

### 2.1 过渡期比例调整

| 规则 | 标准版 | 小资金过渡版 |
|------|--------|-------------|
| **单只个股上陑** | ≤ 4% | ≤ **10%** |
| **个股合计上陑** | ≤ 25% | ≤ **40%** |
| **个股最少数量** | 无硬性要求 | **≥ 3 只，不同行业** |
| **单只最低建仓** | 无 | **≥ 8,000 元** |
| **触发回归条件** | — | **总资产 ≥ 50 万** |

### 2.2 过渡期三层架构建议（以 16.3 万为例）

| 层级 | 金额 | 配置方式 |
|------|------|---------|
| **基石层 30%** | ~4.9 万 | 沪深300ETF + 海外宽基ETF + 黄金ETF |
| **现金流层 35%** | ~5.7 万 | 2-3 只高息股（每只 1-2 万）+ 红利ETF |
| **成长层 20%** | ~3.2 万 | 1-2 只跨市场ETF |
| **机动层 15%** | ~2.4 万 | 货币基金/短债 |

### 2.3 过渡期不变的底层纪律

以下约束**任何时候都不得放宽**：
- 年度储蓄率 ≥ 50%
- 不使用任何杠杆
- 7 天冷静期
- 不追热点、不盯盘
- 成长层利润向上流动
- 任何调整需老大书面确认
- Agent 不主动挖掘个股

### 2.4 回归机制

- 当**总资产 ≥ 50 万**时，自动触发回归标准版比例。
- 回归前 30 天，Agent 须向老大发出警示，并提供回归计划。
- 回归过程分 **3 个月** 完成，逐步减少个股集中度。
