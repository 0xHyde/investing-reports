## 计算逻辑说明

**核心公式：**

- **Fire 目标资产** = 年支出 ÷ 提款率（默认 4%）
- **年资产增长** = 年初资产 × 投资收益率 + 年储蓄
- **达到 Fire** = 当资产 ≥ Fire 目标资产时

**参数默认值：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 月工资 | 15,000 | 税后月收入 |
| 年终奖 | 30,000 | 税后年终一次性收入 |
| 年化收益率 | 7% | 投资组合长期预期回报 |
| 提款率 | 4% | Fire 后每年可安全提取的比例 |
| 收入增长率 | 5% | 工资/收入年增长预期 |
| 支出增长率 | 3% | 通胀导致的支出增长 |

> 💡 **提示**：4% 提款率来自 Trinity Study，假设投资组合以股票为主、30年提取期。如果你的 Fire 时间更早或风险偏好不同，可调整此参数。
