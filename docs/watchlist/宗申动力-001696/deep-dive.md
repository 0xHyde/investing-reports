# 宗申动力(001696) 深度分析报告

## 一、叙事书：公司故事与产业逻辑

### 1.1 赛道定位——低空经济"心脏"供应商

宗申动力是国内通用小型汽油机及摩托车发动机的传统龙头，近年战略转型的核心是：**从地面动力向空中动力跃迁**。公司依托宗申集团四十年制造业积淀，切入低空经济最硬核的环节——**航空发动机及通航动力系统的国产化替代**。

低空经济已被写入国家战略新兴产业规划。2024-2025年，低空空域管理改革加速，无人机物流、农林植保、应急救援、城市空中交通(UAM)等场景进入商业化前夜。而发动机的可靠性、功率密度、油耗水平，直接决定低空飞行器的性能天花板。宗申动力卡位"卖铲人"角色，逻辑极硬。

### 1.2 竞争格局——国内稀缺，对标海外百年巨头

通用航空动力市场长期被**美国Lycoming、Continental，奥地利Rotax**等外资品牌垄断，国产替代空间巨大。

国内同行中：
- **航发动力(600893)**：聚焦军用大推力航发，通航小活塞发动机非其重点
- **航发控制(000738)**：侧重控制系统，不直接生产整机动力
- **宗申动力**：**国内少有的具备量产能力的通航活塞发动机民企**，产品线覆盖50-300马力区间，适配轻型飞机、直升机、大型无人机

公司旗下**宗申航发**已完成多款航空活塞发动机定型，部分型号已取得民航局适航认证（CAAC TC/PC）。在军用大型无人机动力领域，也已实现批量配套。

### 1.3 公司故事——老制造业的二次起飞

宗申集团的创始人左宗申从修摩托车起家，将宗申动力打造成中国摩托车动力第一品牌。2016年后，摩托车行业见顶，公司果断转型：

- **第一步**：收购加拿大Ducati Energia部分资产，获取航空发动机技术
- **第二步**：成立宗申航发，独立运营航空业务
- **第三步**：2023-2024年，多款产品进入小批量交付，2025年开始放量

2025年年报和2026年一季报双双高增，验证转型进入收获期。航空动力业务占比持续提升，从"概念"走向"业绩兑现"。

### 2.2 成长性分析

- **营收增速**：2025年+28.9%，2026Q1进一步加速至+39.1%，呈现逐季提速态势
- **利润增速**：2025年+27.5%，2026Q1跳升至+88.3%，利润弹性远大于营收弹性，说明高毛利航空业务占比提升+规模效应显现
- **季度利润年化**：Q1单季2.26亿，简单年化约9亿，对应远期PE约17.6倍，当前PE 33.4倍隐含市场对其高成长持续性的认可

### 2.3 盈利能力与运营效率

- **毛利率13.6%**：传统通机业务毛利率偏低（约10-12%），航空发动机毛利率可达25-35%，未来毛利率有明确上行空间
- **ROE 9.45%**：处于制造业中等偏上水平，航空业务占比提升后，ROE有望向15%以上迈进
- **Q1 ROE 4.44%**：若全年维持此水平，年化ROE接近18%，表明2026年可能是ROE跃升的关键年份

## 四、配置建议

### 4.1 投资评级：**增持**（成长层核心仓位）

### 4.2 核心逻辑
1. **赛道红利**：低空经济是未来5-10年确定性最高的新兴产业之一，发动机是产业链"咽喉"
2. **国产替代**：通航动力国产化率不足10%，替代空间巨大
3. **业绩拐点**：2025-2026年业绩连续加速，Q1利润+88%确认高成长
4. **估值合理**：PEG约1.2，远未透支远期空间

### 4.3 目标价位
- **乐观情景（2026E利润9亿，PE 35x）**：目标价约28元，上涨空间 **+57%**
- **中性情景（2026E利润7.5亿，PE 30x）**：目标价约20元，上涨空间 **+12%**
- **悲观情景（业绩不及预期，PE 25x）**：支撑位约14-15元

### 4.4 操作建议
- **已有持仓**：持有为主，Q1业绩超预期后可适当加仓
- **新建仓位**：建议分批建仓，17元以下安全边际较高
- **仓位建议**：成长层核心仓位，单票占比不超过组合的8-10%
- **跟踪要点**：2026年中报航空业务营收占比、宗申航发新增订单、适航认证进展
