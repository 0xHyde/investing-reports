## III. 叙事书：三问

#### Q1：五年后，阳光电源会成为什么样子？

**产业逻辑**：全球能源转型不可逆，光伏+储能从"配角"走向"主力电源"。逆变器是光伏系统的"心脏"，储能系统是新能源消纳的"刚需"。阳光电源是全球逆变器出货量第一、储能系统出货量前列的龙头企业。

**五年图景**：
- **乐观情景**：储能业务占比从当前约30%提升至50%+，海外市场（尤其美国、中东）持续放量，公司成为全球光储一体化解决方案龙头，营收突破1500亿。
- **中性情景**：光伏行业产能过剩持续2-3年，价格战压缩组件端利润，但逆变器/储能因技术壁垒维持相对合理利润率，公司营收1000-1200亿，利润增速放缓至10-15%。
- **悲观情景**：全球贸易壁垒升级（美国关税、欧洲本土制造政策），国内价格战延伸至逆变器/储能，技术迭代被颠覆（如组串式被微型逆变器/功率优化器替代），公司陷入增长停滞。

**核心驱动力**：海外储能市场爆发 + 逆变器出海份额提升。

#### Q2：它的成长/竞争优势靠什么驱动？

**利润模式**：
- 逆变器（占收入约40%）：硬件+软件，技术迭代快，品牌溢价明显
- 储能系统（占收入约30%）：集成+EMS，项目制，单个体量大
- 电站开发/EPC（占收入约20%）：现金流重，利润薄
- 其他（氢能等）：早期，贡献小

**护城河类型**：
1. **规模优势**：全球逆变器出货量第一，采购成本、研发投入摊薄优势明显
2. **品牌/渠道**：海外分布式市场品牌认知度高，渠道布局深入
3. **技术迭代**：研发投入行业领先，产品更新周期短
4. **切换成本**：大型电站客户一旦选定逆变器品牌，替换成本高

**护城河趋势**：**稳定偏宽**。逆变器技术迭代快，但阳光持续高研发投入（年研发费用率约5-6%），与追赶者（华为、上能、锦浪）保持1-2代产品差距。

**定价权**：**中等**。逆变器不是完全commodity，但在国内大型地面电站招标中价格竞争激烈；海外分布式和品牌溢价支撑毛利率。

#### Q3：什么情况下这个叙事会证伪？

**红线场景**：
1. **连续两个季度营收/利润同比负增长且毛利率下滑** — 说明价格战已从组件蔓延至逆变器/储能
2. **海外大客户流失** — 美国市场因政策风险（关税、IRA补贴取消）导致订单断崖
3. **技术路线被颠覆** — 如微型逆变器/功率优化器在大型地面电站中替代组串式逆变器
4. **现金流恶化** — 经营现金流持续低于净利润，应收账款和存货大幅攀升

**2026Q1现状**：营收-18%、净利-40%，但毛利率仍维持33.26%高位（仅同比降1.87pct），**尚未触及红线1**。需观察Q2是否回暖。

#### 利润收割与时间止损

- **利润收割触发**：PE > 35x 或 单只浮盈 > 50% → 减仓50%，利润向上流动至核心层
- **时间止损**：持有18个月，若业务未按预期发展（储能占比未提升、海外收入下滑）→ 退出
- **止损纪律**：浮亏-30%无条件止损
- **利润流动**：成长层利润必须向上流动至核心层（招行、红利ETF。沪深300等），不得内滚

#### 反对意见（Agent 必须说的话）

> **最大的风险不是Q1业绩下滑，而是光伏行业的"周期底"可能比你想象的更长。** 2018年"531新政"后，光伏行业花了2年才走出低谷。当前产能过剩比2018年更严重，价格战从硅料、硅片、电池、组件一路打下来，逆变器和储能虽然目前还是"净土"，但历史证明没有永远的净土。阳光电源的海外占比高，这既是护城河也是软肋——美国政策一旦变脸，业绩可能瞬间变脸。22.8x PE在制造业中不算便宜，你是在为"光储龙头"的叙事付溢价，但这个叙事已经部分 priced in。如果一定要买，等105元以下。
