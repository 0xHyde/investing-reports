## 一、叙事书：产业逻辑与竞争格局

### 1.1 产业逻辑：国产替代的"生死时速"

半导体设备是整个芯片产业链的"咽喉"，光刻机、刻蚀机、薄膜沉积设备、量测设备等决定了芯片制造的制程能力和良率。全球半导体设备市场约1200亿美元，中国市场占比超30%，但国产化率不足20%——在光刻机领域几乎为0，在刻蚀、薄膜沉积等部分环节约10-30%。

**国产替代的三重驱动：**
1. **地缘政治倒逼**：美国对华半导体出口管制持续升级，EUV光刻机、先进制程设备全面封锁，自主可控从"可选项"变为"必选项"
2. **内循环支撑**：国内晶圆厂（中芯国际、华虹、长江存储、长鑫存储）扩产加速，2024-2027年是中国大陆晶圆产能建设高峰，设备采购需求刚性
3. **技术突破窗口**：国产设备在成熟制程（28nm及以上）已基本打通，部分环节进入14nm验证，虽然与ASML、应用材料等国际巨头差距仍大，但"能用"到"好用"的迭代正在加速

### 1.2 竞争格局：群雄逐鹿，各守一摊

半导体设备细分领域极多，每个环节都是独立的"技术黑箱"，难以跨界竞争：

| 细分领域 | 国际龙头 | 国内代表 | 国产替代进度 |
|----------|----------|----------|--------------|
| 光刻机 | ASML、Nikon、Canon | 上海微电子 | <5%，差距最大 |
| 刻蚀设备 | 拉姆研究、TEL、应用材料 | 中微公司、北方华创 | 20-30%，进展较快 |
| 薄膜沉积 | 应用材料、TEL、LAM | 北方华创、拓荆科技 | 15-25% |
| CMP设备 | 应用材料、Ebara | 华海清科 | 20-30% |
| 量测/检测 | KLA、Applied Materials | 中科飞测、精测电子 | 10-20% |
| 测试设备 | 泰瑞达、爱德万 | 长川科技、华峰测控 | 20-30% |
| 材料（光刻胶） | JSR、TOK、信越 | 南大光电、彤程新材 | 10-20% |

**ETF持仓核心逻辑：**
持仓覆盖了国产替代进度最快的几个环节——刻蚀（中微公司）、测试（长川科技）、薄膜沉积（拓荆科技、北方华创）、CMP（华海清科）、量测（中科飞测）、光刻胶（南大光电），以及晶圆代工龙头中芯国际、CPU/GPU龙头海光信息作为"设备需求方"和"算力底座"。

## 三、风险扫描

### 🔴 高风险
1. **地缘政治黑天鹅**：美国对华半导体封锁若从"设备禁运"升级为"长臂管辖"（制裁使用美设备的中国晶圆厂），将直接冲击整个行业
2. **技术瓶颈无法突破**：先进制程设备（尤其是光刻机）若长期无法突破，国产替代天花板受限，估值逻辑动摇
3. **半导体周期下行**：全球半导体资本开支具有强周期性，若2025-2026年进入下行周期，设备需求将收缩

### 🟡 中等风险
4. **估值泡沫**：当前45-55倍PE已包含较高预期，若业绩兑现不及预期，估值收缩风险大
5. **个股黑天鹅**：南大光电权重超20%，单一标的风险集中；海光信息若被制裁将重创组合
6. **行业竞争恶化**：国内设备企业一哄而上，部分环节可能出现价格战

### 🟢 低风险但需跟踪
7. **ETF流动性风险**：规模37亿较小，大额进出可能影响价格
8. **晶圆厂扩产不及预期**：若国内晶圆厂因资金/技术原因放缓扩产，设备采购需求下降
