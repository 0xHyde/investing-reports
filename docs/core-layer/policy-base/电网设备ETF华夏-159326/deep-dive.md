## 一、叙事书：产业逻辑与竞争格局

### 1.1 产业逻辑：被忽视的"新基建底座"

如果说新能源发电是新型电力系统的"源"，那电网就是"网"和"储"的物理载体。风光大基地在西北，负荷中心在东部，特高压输电是将清洁电力从西部送往东部的唯一可行方案。同时，分布式光伏、充电桩、储能的爆发式增长，倒逼配电网从"单向输送"升级为"双向互动"的智能电网。

国家电网"十四五"规划投资超2.4万亿，"十五五"大概率只增不减。特高压建设进入第三轮高峰期（2024-2027年），每年核准开工4-6条直流线路。配网智能化改造（智能电表、配电自动化、虚拟电厂）刚刚起步，市场空间千亿级。这是一个**政策刚性驱动、周期长达10年、竞争格局稳定**的黄金赛道。

### 1.2 竞争格局：国家队主导，集中度提升

电网设备行业具有典型的"国家队"特征：
- **特高压/高压领域**：国电南瑞、中国西电、特变电工、平高电气等国企主导，技术门槛高、资质壁垒强，新进入者几乎不可能
- **智能电网领域**：国电南瑞是绝对龙头（调度自动化市占率超70%），四方股份、许继电气跟随
- **线缆/材料领域**：中天科技、亨通光电在光纤光缆+海缆领域双轮驱动，特变电工在变压器/线缆领域全品类覆盖

行业集中度持续提升，头部企业在手订单饱满，2025-2026年是业绩兑现期。

## 三、风险扫描

### 🔴 高风险
1. **投资周期波动**：电网投资受宏观经济和财政状况影响，若经济下行导致投资推迟，业绩兑现延后
2. **特高压核准不及预期**：特高压项目核准节奏存在不确定性，近年存在"核准-开工-投运"周期拉长现象
3. **特变电工硅料拖累**：特变电工多晶硅业务占比高，硅料价格若持续低迷将拉低整体组合表现

### 🟡 中等风险
4. **行业竞争加剧**：配网领域门槛相对较低，中小企业价格战可能侵蚀毛利率
5. **技术替代风险**：固态变压器、柔性直流等新技术路线若超预期发展，可能颠覆现有设备格局
6. **海上风电不及预期**：中天科技、亨通光电海缆业务占比提升，若海风装机放缓将拖累业绩

### 🟢 低风险但需跟踪
7. **利率上行风险**：高股息国企属性资产在利率上行周期估值承压
8. **ETF折溢价**：场内交易价格相对净值出现折溢价波动
